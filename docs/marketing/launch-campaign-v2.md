# ID 도플갱어 Launch Campaign V2

Last updated: 2026-06-11

## Campaign Thesis

**Core idea:** 아이디는 기억보다 오래 남습니다.

ID 도플갱어는 "누군가를 찾는 서비스"가 아니라 "내가 쓰려는 username이 공개 표면에서 어떻게 보이는지 점검하는 서비스"로 포지셔닝해야 한다. 이 차이가 런칭 성공 여부를 가른다. 자극적인 사람 찾기 메시지는 클릭은 만들 수 있어도 앱스토어, 커뮤니티, 언론, 장기 신뢰를 망친다.

## One-Liner Stack

**Public one-liner:** 내 아이디, 어디에 남아 있을까?

**Emotional line:** 아이디는 기억보다 오래 남습니다.

**Safety line:** 사람 찾기가 아니라 username 문자열 점검입니다.

**Creator line:** 활동명 확정 전, 겹치는 공개 흔적부터 확인하세요.

**Brand line:** 브랜드명 공개 전, 공식 계정 선점 리스크를 점검하세요.

## Primary Audience

1. **크리에이터/1인 브랜드:** 활동명, 닉네임, 채널명을 정하기 전 겹치는 공개 흔적을 확인하고 싶다.
2. **초기 스타트업/브랜드:** 서비스명 공개 전에 주요 플랫폼의 username 선점 상태를 알고 싶다.
3. **개인 사용자:** 오래 쓴 아이디가 어디까지 공개 표면에 남아 있는지 정리하고 싶다.
4. **마케팅/브랜드 운영자:** 클라이언트나 캠페인 핸들의 계정 선점과 사칭 신호를 리포트로 정리하고 싶다.

## Launch Offer

**Offer:** 무료 요약 점검으로 공개 흔적을 먼저 확인하고, 필요한 경우 정밀 리포트로 전체 URL·점수·조치 가이드를 저장한다.

**CTA:** 내 아이디 흔적 찾기

**Secondary CTA:** 활동명 겹침 확인

**Do not say:** 숨은 계정 찾기, 동일인 찾기, 사칭범 추적, 개인정보 검색, 계정 주인 확인.

### Product Hunt

**Name:** ID Doppelganger

**Tagline:** Check public username risk without people-search claims

**Description:** ID Doppelganger is a Korean-first username risk checker. Enter a username, nickname, creator handle, or brand name to review public account candidate cards first, then rarity, exposure, impersonation-candidate signals, and abandoned-account risk. It is not a people-search product and does not claim identity matching.

**Maker comment:**

Hi Product Hunt. I built ID Doppelganger because username reuse is a real problem, but this category can get unsafe quickly.

The product is intentionally narrow. It checks public username-string candidates, shows the platform cards first, and turns them into a practical report with rarity, exposure, impersonation-candidate signals, abandoned-account risk, and next-step guidance.

It does not search real names, phone numbers, emails, locations, profile photos, post content, or same-person probability.

The first version is Korean-first. I would value feedback on whether "username risk check" makes the boundary clear enough for creators, founders, and small brands.
