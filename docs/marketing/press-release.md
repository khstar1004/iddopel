## ID 도플갱어 출시, "아이디는 기억보다 오래 남는다"

**서울, 2026년 6월 11일** - ID 도플갱어는 아이디, 닉네임, 활동명, 브랜드명의 공개 username 사용 후보를 먼저 보여주고 희소성, 노출도, 사칭 가능성, 방치 계정 위험을 참고 분석으로 정리하는 한국어 서비스를 출시했다.

많은 사용자는 같은 아이디를 여러 커뮤니티와 SNS에서 반복해서 쓴다. 크리에이터와 초기 브랜드는 활동명이나 서비스명을 공개한 뒤에야 같은 이름의 계정이 이미 존재한다는 사실을 발견하기도 한다. ID 도플갱어는 이 문제를 "사람 찾기"가 아니라 "username 문자열 점검"으로 좁혀 다룬다.

사용자는 점검할 아이디나 브랜드명을 입력하고 정당한 목적을 확인한 뒤, 공개 플랫폼에서 같은 username 후보가 보이는지 바로 확인할 수 있다. 무료 점검은 플랫폼 후보 카드와 잠긴 상세 URL을 먼저 제공하며, 정밀 리포트는 전체 URL, 위험도, 조치 가이드, HTML/PDF 저장 기능을 제공한다.

서비스의 핵심 원칙은 안전한 경계다. ID 도플갱어는 실명, 전화번호, 이메일, 주민등록번호와 유사한 값, 위치 정보, 프로필 사진, 게시글 내용 검색을 지원하지 않는다. 발견된 계정 후보 역시 동일인 여부를 뜻하지 않으며, 공개 username 문자열이 사용된 후보 신호로만 해석하도록 설계됐다.

ID 도플갱어 관계자는 "아이디 재사용과 브랜드 계정 선점 문제는 실제로 불편하지만, 이를 신상 확인이나 사람 추적으로 다루면 위험해진다"며 "출시 전 활동명 점검, 오래된 아이디 정리, 공식 계정 선점, 사칭 후보 검토처럼 합법적이고 제한된 사용 장면에 집중했다"고 말했다.

주요 사용 장면은 다음과 같다.

- 오래된 아이디가 공개 플랫폼에 얼마나 남아 있는지 점검
- 크리에이터 활동명이나 닉네임 후보 확인
- 브랜드명과 서비스명 공개 전 공식 계정 선점 우선순위 정리
- 사칭 후보 계정 검토와 신고 준비
- 디지털 풋프린트 정리 전 공개 username 후보 파악

ID 도플갱어는 웹/PWA 흐름을 우선 제공하며, Android/iOS 앱 제출 준비물과 토스 인앱 경로도 함께 준비되어 있다. 이후 정밀 리포트 결제, 월간 모니터링, 안전한 공유 카드 기능을 단계적으로 확장할 예정이다.

- 희소성, 노출도, 사칭 가능성, 방치 계정 위험 참고 분석
