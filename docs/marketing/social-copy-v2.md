## LinkedIn Variants

### Founder Build

아이디는 기억보다 오래 남습니다.

ID 도플갱어를 만들면서 가장 먼저 정한 원칙은 "추적하지 않는다"였습니다.

우리가 풀고 싶은 문제는 분명합니다. 사람들은 같은 아이디를 오래 쓰고, 크리에이터는 활동명을 브랜드처럼 쓰고, 초기 스타트업은 서비스명을 공개한 뒤에야 계정 선점 문제를 발견합니다.

대신 하지 않는 것도 명확합니다.

실명 검색 없음. 전화번호/이메일 검색 없음. 위치 추론 없음. 사진 분석 없음. 게시글 내용 검색 없음. 동일인 판정 없음.
